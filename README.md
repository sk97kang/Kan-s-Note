# Kan's Note

Make Simple Note and the name is Kan's Note

React + Ttypescript를 이용한 첫 번째 프로젝트

## 사용 기술

- React
- Typescript
- Local Storage

## 기능

- Add Note : 새로운 노트를 추가
- Del Note : 선택한 노트를 삭제
- Edit Note : 선택된 노트의 제목과 내용 수정
- Actived Note : 선택된 노트 강조
- Save Local Storage : 업데이트 시 데이트 세이브
- Load Local Storage : 처음 실행 시 데이터 로드
